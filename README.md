# Nim Bot

General purpose discord bot written in Nim using the dimscord-library.

## Dependencies
- [dimscord](https://github.com/krisppurg/dimscord)
- [dimscmd](https://github.com/ire4ever1190/dimscmd)

## Features:
- !sum [number] ...
- !multiply [number] ...
- magischer linuxmann [question]
- cheg
